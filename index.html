BIENVENIDOS!!

En este sitio Ud. encontrara información que facilitara la comprensión del diseño de sitios Web utilizando herramientas colaborativas como Git y Github. En el medio tecnológico es común que exista la confusión que GitHub es igual a Git lo cual no es cierto. Git es el programa y el protocolo. Y GitHub es uno de los muchos servicios (como BitBucket o GitLab) que te almacenan código. Es posible usar Git de forma local, también es posible implementar tu propio servidor de Git, o también usar un servicio ya existente, pero es importante no confundir GitHub con Git.
